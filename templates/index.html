🔌 API 接口

GET /api/status获取系统状态与当前节点
GET|POST /api/switch随机切换到 Top N 节点
GET|POST /api/switch/<tag>切换到指定节点
GET /api/test?url=<target>切换节点后测试目标 URL，默认 https://api.ipify.org
GET /api/nodes获取所有可用节点及延迟
GET /api/top获取 Top N 节点
GET /api/node/<tag>获取节点完整配置（sing-box 格式）
GET /api/request_logs获取请求日志
GET /api/reload重新加载订阅节点
GET /api/speedtest/<tag>?url=<file>测试指定节点下载速度，可选自定义测速文件
GET /api/speedtest_cache读取最近测速缓存
GET /api/health健康检查，包含代理可用性测试
POST /api/restart_singbox手动重启 sing-box

💡 cURL 示例

# 查看状态
curl http://YOUR_IP:{{ status.web_port }}/api/status

# 随机切换节点
curl http://YOUR_IP:{{ status.web_port }}/api/switch

# 测试出口 IP
curl "http://YOUR_IP:{{ status.web_port }}/api/test?url=https://api.ipify.org"

# 获取节点列表
curl http://YOUR_IP:{{ status.web_port }}/api/nodes

# 手动重启 sing-box
curl -X POST http://YOUR_IP:{{ status.web_port }}/api/restart_singbox

💡 Python 示例

import requests

API = "http://YOUR_IP:{{ status.web_port }}"
{% if status.proxy_auth_enabled %}
PROXY = {"http": "http://USER:PASSWORD@YOUR_IP:{{ status.listen_port }}", "https": "http://USER:PASSWORD@YOUR_IP:{{ status.listen_port }}"}
{% else %}
PROXY = {"http": "http://YOUR_IP:{{ status.listen_port }}", "https": "http://YOUR_IP:{{ status.listen_port }}"}
{% endif %}

status = requests.get(f"{API}/api/status").json()
print(status)

requests.get(f"{API}/api/switch")
r = requests.get("https://api.ipify.org", proxies=PROXY)
print(r.text)

result = requests.get(f"{API}/api/test", params={"url": "https://api.ipify.org"}).json()
print(result)
